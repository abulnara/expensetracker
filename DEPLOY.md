# Budget Tracker — Deployment Guide

## What you have
A complete React app ready to deploy to GitHub Pages for free.
Your live URL will be: https://YOUR-USERNAME.github.io/expensetracker/

---

## Step 1 — Create a GitHub account
1. Go to https://github.com
2. Click "Sign up" and create a free account
3. Verify your email

---

## Step 2 — Create a new repository
1. Click the "+" icon in the top right → "New repository"
2. Name it exactly: `expensetracker`
3. Make sure it is set to **Public**
4. Do NOT check "Add a README file"
5. Click "Create repository"

---

## Step 3 — Upload the project files
1. On your new empty repository page, click "uploading an existing file"
2. Open the `expensetracker` folder on your laptop
3. Select ALL files and folders inside it and drag them into the GitHub upload area
   - Make sure you include: src/, public/, .github/, index.html, package.json, vite.config.js, .gitignore
4. Scroll down, click "Commit changes"

---

## Step 4 — Enable GitHub Pages
1. Go to your repository → "Settings" tab
2. In the left sidebar click "Pages"
3. Under "Source" select "GitHub Actions"
4. That's it — GitHub will now auto-build and deploy your app

---

## Step 5 — Wait ~2 minutes
1. Click the "Actions" tab in your repository
2. You will see a workflow running called "Deploy to GitHub Pages"
3. Once it shows a green checkmark, your app is live

---

## Your app is live at:
https://YOUR-USERNAME.github.io/expensetracker/

Replace YOUR-USERNAME with your actual GitHub username.

---

## Install on your phone (optional)
1. Open the URL in Safari (iPhone) or Chrome (Android)
2. Tap the Share button → "Add to Home Screen"
3. The app will appear on your home screen like a native app

---

## Sharing with others
Just send them the URL: https://YOUR-USERNAME.github.io/expensetracker/
Anyone can use it — no login required.
Each person's data is stored privately on their own device.

---

## Updating the app in the future
1. Edit the files on your laptop
2. Go to your GitHub repository
3. Click on the file you want to update → pencil icon → paste new code → commit
4. GitHub automatically rebuilds and deploys within ~2 minutes
